public ResultSet setHydratedColumns(array $hydrated_columns, array $ignore_duplicate_columns = false)

By using setHydratedColumns() on a resultset, you ensure that only those columns will be available/returned even if they are more available in the original dataset

Checks against column name existence can only be done lazily when current() is called. (in case the resultset does not contain rows, we don't know about columns)

Parameters

array $hydrated_columns columns nams in an array
array $ignore_duplicate_columns check whether we can ignore duplicate columns

Return Value

ResultSet

Exceptions

InvalidArgumentException
DuplicateColumnException only when $ignoreduplicatecolumns is false (default)
