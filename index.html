Welcome to Soluble Components .

Soluble components is a set of PHP5.3+ components easily injectable in your project. Soluble is written in PHP 5.3 using namespaces and following PSR-2 recommendation. The code is hosted on GitHub and available on packagist.

Documentation

The documentation is hosted on http://soluble-components.readthedocs.org/en/latest/
